# Инвестиционный калькулятор

Веб-приложение для расчёта роста инвестиционного портфеля с учётом сложного процента.

## Описание

Инструмент позволяет моделировать доходность инвестиций с настраиваемыми параметрами и наглядно отображает результаты в виде графика и таблицы по годам.

## Возможности

- Указание начальной суммы и ежемесячных пополнений
- Настройка годовой доходности (ползунок + поле ввода)
- Выбор срока инвестирования от 1 до 100 лет
- Выбор периодичности капитализации: ежемесячно, ежеквартально, раз в полгода, ежегодно
- Отображение итоговых показателей: итоговый капитал, сумма взносов, доход, доходность на вложения
- Визуализация структуры капитала (взносы vs доход) в виде прогресс-бара
- Интерактивный столбчатый график роста капитала (Chart.js)
- Таблица с разбивкой по годам

## Технологии

- HTML5 / CSS3 / Vanilla JavaScript
- [Chart.js 4.4](https://www.chartjs.org/) — построение графиков

## Запуск

Откройте файл `index.html` в любом современном браузере. Никаких зависимостей для установки не требуется — приложение полностью работает на стороне клиента.
